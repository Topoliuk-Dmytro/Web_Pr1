# Веб-калькулятор палива — Практична робота №1

Веб-застосунок на Go для розрахунку складу сухої та горючої маси палива та нижчої теплоти згоряння.

## Вимоги

- [Go 1.21+](https://go.dev/dl/)

## Запуск

```bash
go run .
```

Або збірка та запуск:

```bash
go build -o fuel-calc.exe .
./fuel-calc.exe
```

Відкрийте в браузері: **http://localhost:8080**

## Завдання

### Завдання 1
Розрахунок складу сухої та горючої маси палива та нижчої теплоти згоряння для робочої, сухої та горючої маси за заданим складом компонентів (Hₚ, Cₚ, Sₚ, Nₚ, Oₚ, Wₚ, Aₚ).

### Завдання 2
Перерахунок елементарного складу та нижчої теплоти згоряння мазуту з горючої маси на робочу.

## Варіант 2 (табл. 1.3)

- Hₚ=4,2%; Cₚ=62,1%; Sₚ=3,2%; Nₚ=20,6%; Oₚ=6,4%; Wₚ=0,7%; Aₚ=2,8%
